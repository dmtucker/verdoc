CLI¶

verdoc¶

Get REFNAMES from --repo and build them into --dest.

verdoc [OPTIONS] [REFNAMES]...

Options

--build-opt <build_opt>¶

Pass keyword arguments to the builder (e.g. foo=bar).

--dest <dest>¶

Specify the path to build in.

Default

current working directory

--log-level <log_level>¶

Specify how verbose output should be.

Default

info

Options

debug | info | warning | error | critical

--repo <repo>¶

Specify the repository to look for references in.

Default

current working directory

--version¶

Show the version and exit.

Arguments

REFNAMES¶

Optional argument(s)

verdoc-index¶

Create an index.html file in --dest that redirects to URL.

verdoc-index [OPTIONS] URL

Options

--dest <dest>¶

Specify the path to build in.

Default

current working directory

--version¶

Show the version and exit.

Arguments

URL¶

Required argument
